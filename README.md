# ADA Assignment

## Algorithms Implemented
- Divide & Conquer: Binary Search, Merge Sort, Quick Sort
- Greedy: Knapsack, Activity Selection, Huffman Coding
- Dynamic Programming: LCS, 0/1 Knapsack
- TSP

## How to Run
python divide_and_conquer.py
python greedy_algorithms.py
python dynamic_programming.py
python tsp.py

## Observations
- Merge sort consistent performance
- Quick sort faster but unstable
- DP gives optimal solution
- TSP shows exponential growth
